# GitHub リポジトリ一覧へのショートカット拡張機能

GitHubのリポジトリページから、そのユーザーのリポジトリ一覧に一発で移動できるブラウザ拡張機能です。

## 機能

- GitHubのリポジトリページ（`https://github.com/username/repo`）を開いている時、ページ上部に「📦 リポジトリ一覧」ボタンが表示されます
- このボタンをクリックすると、そのユーザーのリポジトリ一覧ページ（`https://github.com/username?tab=repositories`）に直接移動できます
- 毎回ユーザー名をクリックしてからリポジトリ一覧を開く手間が省けます

## インストール方法

1. `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択

## 使用方法

1. GitHubの任意のリポジトリページを開きます（例：`https://github.com/username/repo`）
2. ページ上部に「📦 リポジトリ一覧」ボタンが表示されます
3. ボタンをクリックすると、そのユーザーのリポジトリ一覧ページに移動します
